実験 I

各項のツマミを回す — 1次元で波形がどう変わるか

1次元の速度分布 u(x) に対して、方程式の各項の「効き具合」をバーで変えられます。対流だけにすると波が突っ立ち（衝撃波）、粘性だけにするとなだらかに拡散します。

u(x) 対流の寄与 粘性の寄与 圧力の寄与 外力の寄与

観察のポイント：対流項は波の高い所ほど速く進ませるので前面が切り立つ（非線形性）。粘性項は曲率 ∂²u/∂x² に比例して凸凹をならす。両方あるときの釣り合いが Re の意味です。

実験 II

2次元の流れをかき混ぜる — 項をON/OFFする

キャンバス上をドラッグすると力を加えて煙（高温ガス）を注入できます。項のON/OFFに加えて、浮力（外力項）で自然対流プルームを起こしたり、表示を渦度 ω に切り替えて渦の回転向きを見ることもできます。

試すこと：①対流OFF → 煙が運ばれず、その場で滲むだけ。②粘性を最大 → 蜂蜜のように渦がすぐ死ぬ。③圧力OFF → 流体が湧いたり潰れたり、「非圧縮」が破れて煙がダマになる。④熱源ON＋浮力を上げる → 温度差だけで上昇流（自然対流プルーム）が生まれる。ろうそくの炎が上に伸びる理由。⑤渦度表示に切り替え → 反時計回り（橙）と時計回り（青）の渦対がプルームの両脇に見える。

実験 III

定義を体験する

物質微分 D/Dt — 「乗って測る」と「立って測る」

温度場 T(x) は時間変化しない定常場（∂T/∂t = 0）。それでも流れに乗った粒子の温度計は変化する。これが移流 (u·∇)T。

u = 0 にすると 2 本の線が一致します。DT/Dt = ∂T/∂t + u·∇T の第2項が「乗っているから見える変化」。運動方程式の左辺 Du/Dt もまったく同じ構造です。

連続の式 ∇·u = 0 — 入った分だけ出ていく

検査体積の各面の流速をバーで変えて、出入りの収支（発散）を体感する。非圧縮流ではこれが常にゼロ。
∇·u = (uR−uL) + (vT−vB) = 0.00　

実験IIで圧力項をOFFにしたときに破れていたのが、この収支です。圧力は「収支を合わせる係」として瞬時に決まります。

レイノルズ数 Re = UL/ν — 対流項と粘性項の綱引き
