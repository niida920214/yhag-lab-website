学術的背景 — なぜ自己相関経由のFFTを用いるのか

1. 解析パイプライン

各フレームのROI内グレースケール輝度 I(x, y) から実効値（RMS）を求め、時系列 x(t) を構成します。

xRMS = √ 1W · H Σx,y I 2(x, y)

平均値を差し引いた輝度変動成分 x'(t) = x(t) − ⟨x⟩ に対して解析を行います。

2. 自己相関関数とウィーナー・ヒンチンの定理

自己相関関数 R(τ) はFFTを介して効率的に計算されます（Wiener–Khinchin の定理）。

R(τ) = ℜ [ IFFT( |FFT(x')|2 ) ],   R̂(τ) = R(τ)R(0)

3. 直接FFTに対する優位性

  • 直接FFTでは、火炎の不規則な揺らぎやランダムノイズがスペクトル全体に分散して乗るため、支配的ピークが埋もれやすくなります。
  • ランダム成分は位相が揃わないため、自己相関関数上ではラグ τ の増加とともに速やかに減衰します。一方、規則的（コヒーレント）な周期振動成分は τ が大きくなっても相関を保ちます。
  • したがって正規化自己相関 R̂(τ) に対して再度FFTを施すと（パワースペクトルのスペクトルに相当する操作）、火炎の支配的な自励振動周波数が極めてシャープかつ高S/N比で抽出されます。
fpeak = arg maxf > 0 | FFT( R̂(τ) ) |2

4. サンプリングと折り返し（エイリアシング）

検出可能な周波数上限はナイキスト周波数 fN = 1 / (2·dt) です。真の振動周波数が fN を超える場合、スペクトル上では低周波側に折り返された偽ピークとして現れるため、高速度カメラのフレームレート（dt）の妥当性を必ず確認してください。
